# Product Demo Outline

A demo cheat-sheet for Ledgerium AI. The core arc is the same every time; the
persona bullets are what to lean into (and what to skip).

## Core demo arc (~10–12 min)

1. **The hook (30s)** — "Your SOP says 5 steps; your team takes 17." Documentation is stale, manual, and unverifiable.
2. **Record a workflow (the magic)** — open the extension, click Record, run a real browser process, Stop. Stress: *no screenshots, no keystrokes, no screen recording* — structured events only.
3. **Instant SOP** — a complete, step-by-step SOP appears, every step traced to an observed event. Generate a **share link** (no login to view).
4. **The workflow library** — the dashboard: portfolio timestudy band + health scores + cycle time + run counts. The line: **"measured, not estimated."**
5. **Process intelligence** — open a workflow → process map (frequency ↔ performance), **variants** (dominant / rework / exception paths), the bottleneck step.
6. **The Report + ROI** — verdict → scorecard → step-duration timestudy → **before/after compare → the ROI number** (arithmetic over *their* observed durations, not a guess).
7. **The moat close + CTA** — "every number traces to a real recording" (deterministic, evidence-linked, privacy-by-architecture). "Record your first workflow free."

> Golden rule: **let the recording do the talking.** Record something *their* team actually does, live, if you can.

## Persona tailoring (2 things to emphasize each)

- **Ops Manager (primary)** — Library health scores to triage which processes are messy; the before/after ROI number to justify the change.
- **Team Lead** — Generate-and-share an SOP for onboarding; variants view showing how differently people run the "same" process.
- **Compliance / Audit** — Evidence trace from any step back to the source event (immutability/audit trail); "no estimates, every figure is observed."
- **CISO / IT Security** — What's *not* captured (no screenshots/keystrokes/content); deterministic processing — no black-box AI running on their process data.
- **IT / CIO** — Zero-integration capture (just a Chrome extension); the process map as a system-of-record for internal tooling.
- **Automation Analyst / AI Builder** — Automation-opportunity scoring to pick targets; the structured process definition as ready input for an automation/AI agent ("measure before you automate").
- **Lean / Continuous Improvement** — Cycle-time timestudy + bottleneck Pareto (a real time-study, automatically); the rework/exception variant paths as waste to eliminate.
- **Internal Consultant / COO** — Library-wide portfolio band as an exec view of where time goes; the Report as a client-ready deliverable in minutes vs. days.
- **Business Analyst** — Variants + DFG to map as-is vs. intended process; exportable, evidence-backed documentation that won't drift.
- **Founder / SMB** — Speed: real SOP in ~60 seconds, free; shareable link so the whole (small) team is on the same page instantly.
