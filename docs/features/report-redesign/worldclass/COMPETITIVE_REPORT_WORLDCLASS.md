## Dashboard/report product STRATEGY (best-in-class analytics, 2026)
The structural shift is **dashboard → decision briefing**: (1) report-as-narrative (verdict → 3-5 numbers → one
hero chart → interpretation → drill); (2) **AI generates the executive summary** (Power BI Smart Narrative per
visual; Amplitude agentic, ~25% of queries; Looker NL narratives); (3) progressive disclosure (30s → 3min → drill);
(4) **scheduled delivery** (Stripe Sigma, Celonis — table stakes by 2026); (5) shareable URL + PDF snapshot; (6)
frontier = **proactive/agentic** surfacing (report arrives when the system finds something).

## Capability matrix (condensed) — Ledgerium vs Celonis/UiPath/Signavio/Apromore/IBM
**Ledgerium LEADS (unique):** evidence-linked figures (every number traces to a recorded event) · observed-only

recordings · figure-level determinism (byte-reproducible, vs competitors' generative summaries) · reaches log-less

**Frontier we LACK:** what-if simulation · benchmarks · NL query ("ask the report") · agentic/proactive delivery.
