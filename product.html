Recorder • Session JSON • Deterministic derivation

Product

The calm recorder that captures real work, plus a schema and rules that produce reproducible maps and SOPs.

Product overview

Ledgerium AI turns user behavior into traceable process intelligence—without losing the raw truth of what happened.

Recorder

Captures browser workflow events (nav/click/input/submit) with timestamps and target context.

  • Start/stop with an explicit REC state
  • Session markers for “job changes”
  • Selective capture + redaction rules

Evidence Drawer

A dual-view timeline that shows human-readable steps and the underlying JSON evidence.

  • Event IDs, targets, and timing
  • Link every SOP line to evidence
  • Export raw + derived objects

Deterministic derivation

Rules translate raw events into normalized steps and a process graph.

  • Same input → same output
  • Published derivation rules
  • Versioned rulesets

Process maps

Generate legible maps from sessions and compare across runs to find variation.

  • Segment into activities
  • Detect decisions/branches
  • Measure time + variation

SOP rendering

Render standard work that stays connected to evidence and updates deterministically.

  • Purpose / scope / prerequisites
  • Steps w/ evidence links
  • Controls + exceptions

Example: step object

{
  "step_id": "stp_0004",
  "label": "Submit for approval",
  "kind": "action",
  "evidence": {
    "event_ids": ["evt_031", "evt_032"],
    "targets": ["button#submit"]
  },
  "timing_ms": { "min": 210, "median": 480, "max": 1200 }
}
