Static multi-page product marketing website. No build step.

## Pages
- Home (`index.html`)
- Product (`product.html`)
- Pricing (`pricing.html`)
- Trust (`trust.html`)
- Docs (`docs.html`)
- Blog (`blog.html` + 3 sample posts)
- About (`about.html`)
- Contact (`contact.html`)
- Security (`security.html`)
- Privacy (`privacy.html`) — template
- Terms (`terms.html`) — template
- 404 (`404.html`)

## Publish on GitHub Pages
1. Create a repo (e.g. `ledgerium-ai-site`)
2. Upload the contents of this zip to the repo root
3. GitHub → Settings → Pages
4. Source: Deploy from a branch
5. Branch: `main` / folder: `/ (root)`
6. Save

## Customize
- Replace placeholders in `robots.txt` and `sitemap.xml`
- Update copy in each HTML page
- Replace `assets/img/logo.svg` with your final logo/icon
- Replace the `mailto:` in `contact.html` with Formspree / Netlify Forms / your own endpoint
