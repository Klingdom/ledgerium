What you get

The MVP focuses on three things: recorder, schema, deterministic derivation.

1) Sidebar Recorder

A calm UI that makes recording explicit and understandable.

  • Clear recording indicator
  • Session markers for segmentation
  • Configurable redaction rules

2) Session JSON

Immutable raw events + derived step objects, kept separate for auditability.

  • Raw events never rewritten
  • Derived objects are reproducible
  • Export anytime

3) Maps & SOPs

Rules-based generation produces consistent diagrams and procedures.

  • Segmentation thresholds
  • Decision-point detection
  • SOP renderer with evidence links
