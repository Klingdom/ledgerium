# Agent Quality Gate

Use this rubric before marking work complete.

## Minimum Bar
- No critical missing upstream artifact
- Average score >= 4.0/5
- Confidence >= 4/5
- Known gaps called out explicitly
- Next steps or owner/status language included where relevant

## Rework Triggers
- Average score < 4.0
- Artifact purpose is unclear
- Missing baseline/target for metrics work
- Missing happy path and edge case coverage in validation work
- Missing explicit request/response behavior in API work
